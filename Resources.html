一些你可能需要的软件

呵，不过是安利一些好用的软件罢了
Bandizip
你只需要这一个解压软件

别用什么WinRAR 了，又难用还简陋，下一个 Bandizip 一用一个不吱声
Windows 无脑下，免费的
Mac 上好像是收费的

安装之后别忘了在上下文菜单那里选中智能解压，而不是默认的打开文件夹

官网下载
PotPlayer
你只需要这一个影音播放器

伟大，无需多言
又专业，又好用
功能自己发掘吧，老多了
Windwos 端免费的，无脑下
Mac 上好像是收费的，不过 Mac 端的播放器有其他推荐

可以选择性的安装额外的编码器，解一些编码格式用

官网下载
uTools
一定程度上替代了 Windows 自带的傻逼搜索

uTools，我们 Windows 端自己的搜索框
快捷键就是同键位：Alt 加空格
里面也有社区可以装各种好玩的插件，默认功能就是搜索

安装之后别忘了把悬浮窗和超级面板关了

官网下载
qBittorrent
你只需要这一个磁力链下载软件

你可能或多或少听说过它
但无所谓，今天开始你就知道了，这就是磁力链 P2P 下载平台的鼻祖
作为一个开源软件，我这里只给出了一个官方版本，有一些第三方的魔改版本据说内容库更多
不过我没法推荐，这里只挂出来原版

官网下载
阿里云盘
个人觉得很好用的网盘

目前网盘无非几家：
靠很多遗老和老资源撑起来的百度网盘，但你也知道，界面廉价的要死，用着就不舒服
以免费和好用著称的蓝奏云，不过同样的，界面不是很好看
如果说百度云是一个 2t 的机械硬盘，那阿里云盘就是个 256g 的 SSD。主打一个不充会员也不限速，当然我还是冲了会员

官网下载
Snap Hutao
原神的多功能集成工具箱

好用就一个字，我只说一次
在 Snap Hutao 出来之前，原神 PC 端想要破解帧数，需要每个版本都去 Github 的一个库里去更新，很麻烦
Snap Hutao 出来之后，原神官方启动器就没打开过了
而且还集成了一键导出抽卡记录，直播兑换码，和各种实时卡池信息

非管理员打开来设置游戏路径
设置路径之后就切成永远管理员打开，开高刷需要管理员模式
